    python, 编程语言

    ​ Python由荷兰数学和计算机科学研究学会的吉多·范罗苏姆于1990年代初设计，作为一门叫做ABC语言的替代品。 Python提供了高效的高级数据结构，还能简单有效地面向对象编程。Python语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的编程语言， 随着版本的不断更新和语言新功能的添加，逐渐被用于独立的、大型项目的开发。

    vscode环境配置

    安装miniconda

    miniconda基本使用：

    操作 命令 说明
    查看所有环境 conda env list 当前已有的环境列表，带 * 号的是当前激活的环境。
    创建新环境 conda create -n 环境名 python=版本号 例如：conda create -n data_analysis python=3.9。
    激活环境 conda activate 环境名 激活后，命令行前面会出现 (环境名)，此时安装的包都会在这个环境里。
    退出环境 conda deactivate 退出当前环境，回到基础环境。
    删除环境 conda remove -n 环境名 --all 彻底删除某个不用的环境。

    conda进行包管理：

    功能分类 命令示例 作用说明
    🔍 查找与查看 conda list 最常用：列出当前环境中已安装的所有包及其版本。
    conda search <包名> 在仓库中搜索包，查看有哪些版本可用（例如：conda search pandas）。
    conda list <包名> 只查看某个特定的包是否已安装及其版本。
    ⬇️ 安装与下载 conda install <包名> 最常用：在当前激活的环境中安装指定包（例如：conda install numpy）。
    conda install <包名>=<版本号> 安装指定版本的包（例如：conda install tensorflow=2.6.0），用于复现旧项目。
    conda install <包名1> <包名2> 同时安装多个包，Conda 会自动解决它们之间的依赖关系。
    conda install -c conda-forge <包名> 从指定的通道（Channel）安装包。如果官方源找不到包，通常去 conda-forge 社区源找。
    conda install -n <环境名> <包名> 给未激活的指定环境安装包（不用切来切去，很方便）。
    ⬆️ 更新与升级 conda update <包名> 将某个包更新到当前 Python 版本兼容的最新版本。
    conda update --all 更新当前环境中所有可更新的包（慎用，可能会导致版本冲突）。
    conda update conda 更新 Conda 工具本身（建议在 base 环境下执行）。
    🗑️ 卸载与清理 conda remove <包名> 卸载当前环境中的指定包。
    conda remove <包名> --all 卸载指定包及其依赖它的其他包。
    conda clean --all 强力推荐：清理下载缓存和无用的包文件，释放磁盘空间（定期执行）。
    📝 导出与复现 conda env export > environment.yml 将当前环境的配置导出为 yml 文件，方便分享给别人或备份。
    conda env create -f environment.yml 根据导出的 yml 文件，一键重建一个一模一样的环境。

    变量

    变量的命名规则：

    • 变量名必须以字母或下划线字符开头
    • 变量名称不能以数字开头
    • 变量名只能包含字母数字字符和下划线（A-z、0-9 和 _）

    字符串方法

    所有字符串方法都返回新值。它们不会更改原始字符串。

    频率 方法 实战描述与典型场景
    ⭐⭐⭐⭐⭐ split() 数据清洗必备。将字符串按分隔符切分为列表（如处理CSV行、解析URL参数）。
    ⭐⭐⭐⭐⭐ strip() 输入处理必备。去除首尾空格或换行符（如处理用户输入、读取文件行）。
    ⭐⭐⭐⭐⭐ replace() 文本修正必备。替换指定子串（如敏感词过滤、路径修正）。
    ⭐⭐⭐⭐⭐ lower() / upper() 标准化必备。统一转为小写或大写（如比对用户名、密码、验证码时忽略大小写）。
    ⭐⭐⭐⭐⭐ in / not in 逻辑判断必备。检查子串是否存在（如 if "error" in log:）。
    ⭐⭐⭐⭐ join() 拼接必备。将列表高效合并为字符串（如生成SQL语句、拼接路径）。
    ⭐⭐⭐⭐ count() 统计必备。统计子串出现次数（如词频统计）。
    ⭐⭐⭐⭐ startswith() / endswith() 路由与过滤必备。判断文件后缀（如 .csv）或URL前缀（如 /api）。
    ⭐⭐⭐⭐ find() 查找必备。查找子串位置，找不到返回-1（比index更安全，不会报错）。
    ⭐⭐⭐ format() 模板化必备。格式化字符串（虽然现在流行 f-string，但 format 在复杂模板中仍常用）。
    ⭐⭐⭐ index() 定位必备。查找子串位置，找不到会报错（需配合 try-except 使用）。
    ⭐⭐⭐ isdigit() 校验必备。判断是否全为数字（如校验金额、ID）。
    ⭐⭐⭐ isalnum() 校验必备。判断是否仅含字母和数字（如校验用户名、密码强度）。
    ⭐⭐⭐ lstrip() / rstrip() 精细清洗。分别去除左侧或右侧空白（如处理缩进代码块）。
    ⭐⭐ encode() / decode() 编码处理。处理字节流与字符串转换（如处理网络请求、文件读写编码）。
    ⭐⭐ title() / capitalize() 格式化。首字母大写（如生成报表标题、人名标准化）。
    ⭐⭐ center() / ljust() / rjust() 排版。文本对齐填充（如打印控制台表格、生成固定宽度日志）。
    ⭐⭐ partition() 拆分。将字符串分为“前-中-后”三部分（比split更适合只切一刀的场景）。

    List（列表）

    Python 编程语言中有四种集合数据类型：

    • 列表（List）是一种有序和可更改的集合。允许重复的成员。
    • 元组（Tuple）是一种有序且不可更改的集合。允许重复的成员。
    • 集合（Set）是一个无序和无索引的集合。没有重复的成员。
    • 词典（Dictionary）是一个无序，可变和有索引的集合。没有重复的成员。

    选择集合类型时，了解该类型的属性很有用。

    为特定数据集选择正确的类型可能意味着保留含义，并且可能意味着提高效率或安全性。

    difference() 返回包含两个或更多集合之间差异的集合。
    difference_update() 删除此集合中也包含在另一个指定集合中的项目。
    discard() 删除指定项目。
    intersection() 返回为两个其他集合的交集的集合。
    intersection_update() 删除此集合中不存在于其他指定集合中的项目。
    isdisjoint() 返回两个集合是否有交集。
    issubset() 返回另一个集合是否包含此集合。
    issuperset() 返回此集合是否包含另一个集合。
    pop() 从集合中删除一个元素。
    remove() 删除指定元素。
    symmetric_difference() 返回具有两组集合的对称差集的集合。
    symmetric_difference_update() 插入此集合和另一个集合的对称差集。
    union() 返回包含集合并集的集合。
    update() 用此集合和其他集合的并集来更新集合。

    迭代器

    生成器

    在 Python 中，使用了 yield 的函数被称为生成器（generator）。

    跟普通函数不同的是，生成器是一个返回迭代器的函数，只能用于迭代操作，更简单点理解生成器就是一个迭代器。

    在调用生成器运行的过程中，每次遇到 yield 时函数会暂停并保存当前所有的运行信息，返回 yield 的值, 并在下一次执行 next() 方法时从当前位置继续运行。

    调用一个生成器函数，返回的是一个迭代器对象。

    匿名函数

    Python 使用 lambda 来创建匿名函数。

    所谓匿名，意即不再使用 def 语句这样标准的形式定义一个函数。

    • lambda 只是一个表达式，函数体比 def 简单很多。
    • lambda 的主体是一个表达式，而不是一个代码块。仅仅能在 lambda 表达式中封装有限的逻辑进去。
    • lambda 函数拥有自己的命名空间，且不能访问自己参数列表之外或全局命名空间里的参数。
    • 虽然 lambda 函数看起来只能写一行，却不等同于 C 或 C++ 的内联函数，后者的目的是调用小函数时不占用栈内存从而增加运行效率。

中值定理

涉及函数的中值定理

  • 有界和最值定理（直接就是求函数的最大值和最小值）
  • 介值定理（介于函数两个之间的函数值，函数的连续性）
  • 平均值定理（多个自变量的介值定理）
  • 零点定理（两个函数值异号得零）

涉及导数的中值定理

  • 费马定理（可导取极值的驻点）！费马定理的证明

  • 罗尔定理（三个条件，1闭区间上连续 2开区间上可导 3端点函数值相等 得 端点间一值导数为0）

    • 构造函数

  • 拉格朗日中值定理（闭区间连续，开区间可导）

  • 柯西中值定理（两个函数之间的定理）

  • 泰勒公式

    • 带拉格朗日余项的泰勒公式
    • 带佩亚诺余项的泰勒公式
      • 麦克劳林公式

  • 积分中值定理（拉格朗日的变种）

  • 新加 导数介值定理

  • 单调性（用于证明根的唯一性）
  • 罗尔原话（n阶方程至多有k个根，则f=0至多有k+n的根）
  • 实系数奇次方程至少有一个实根

微分不等式

  1. 用函数性态（单调性，凹凸性，最值）
  2. 用常数变量化证明不等式
  3. 用中值定理（主要是拉格朗日和泰勒公式）

一元函数积分学的概念和计算

不定积分

  1. 原函数和不定积分（不定积分没有上下限）
  2. 原函数（不定积分存在定理）
    1. 连续函数必有原函数
    2. 含有第一类间断点和无穷间断点的函数在包含该间断点的区间内没有原函数（含有震荡间断点的函数也不一定有原函数）

定积分

  1. 定积分的概念（将不规则图形划分为规则的梯形）

    • 积分区间有限
    • 被积函数有界

  2. 定积分存在定理

    1. 定积分存在的充分条件
      1. f在闭区间a,b上连续，原函数存在
      2. f在闭区间上单调，原函数存在
      3. f在闭区间上有界，且有有限个间断点（可以从间断点处拆开），原函数存在
    2. 定积分存在的必要条件
      1. 若定积分存在，其在上下界上有界

  3. 定积分的性质

    1. 求区间长度
    2. 积分中的常数可以提到积分符号之前
    3. 积分可加/拆分性
    4. 积分的保号性（积分的绝对值小于绝对值的积分）
    5. 估值定理（积分的介值定理）
    6. 积分中值定理

变限积分

  1. 变限积分的定义
  2. 变限积分的性质
    1. f在a,b上可积，则f的积分在a,b上连续（积分区域的增加是连续的）
    2. f在a,b上连续，则f的积分在a,b上可导（积分区域的增加是平滑的）
  3. 变限积分的求导公式

反常积分

  1. 反常积分的概念（1 积分区间无线，2 被积函数无界）

    • 被积函数越小，反常函数通常越容易收敛
    • 被积区间无线的反常函数收敛，其被积函数不一定收敛于0，可能是被积函数的无穷小程度超过了高的无穷大程度

  2. 无穷区间上反常函数的概念和敛散性

    1. 上积分区间无限
    2. 下积分区间无限
    3. 上下都无限

  3. 无界函数的反常积分概念和敛散性

    1. 积分后从被积函数的瑕点趋近

  4. 题型：

    1. 反常积分判敛公式

积分计算

不定积分的积分法

  1. 基本积分公式

  2. 凑微分法（将被积函数中的一部分提取到积分最右）

  3. 换元法（当被积函数不容易积分的时候（反对幂函指三））

    1. 三角函数代换（三种平方形式）
    2. 恒等变形后做三角变换（先化成三角函数代换方式在代换）
    3. 根式代换（根式内无法凑平方的，直接将整个根式代换）
    4. 倒代换（被积函数分母的幂次比分子高两次或者以上的时候）
    5. 复杂函数直接代换（反对幂函指三）

  4. 分部积分法（分部积分的推广公式）

    1. 在积分中凑出于前式一样相减消去

  5. 有理函数的积分（拆分为若干有理项之和）

定积分的计算

主要使用莱布尼兹公式

含有断点的函数也有可能有原函数（从断点处断开）

  1. 定积分的换元积分（注意换元之后的上下限也要变化）
  2. 定积分的分部积分法
  3. 题型：
    1. 定积分的精确定义
    2. 定积分的计算
      1. 奇偶性
      2. 周期性
      3. 区间再现公式
      4. 华理士

多元微分法

基本概念

  1. 平面点集的基本概念
    1. 内点
    2. 外点
    3. 边界点
  2. 极限（函数从任意方向趋近，目前只用来证明不可导）

  5. 可微（可微的公式）
  6. 偏导数的连续性（在瑕点处定义法求，在非瑕点处公式求）

多元函数求导规则

  2. 隐函数存在定理（一元函数和二元函数）

多元函数的极最值

  1. 概念
  2. 无条件极值
    1. 二元函数取极值的必要条件（一阶偏导存在且取极值，导数为0）
    2. 二元函数取极值的充分条件（对函数求三个偏导，记住公式）
    3. 条件极值和拉格朗日乘数法（记住公式）
  3. 题型
    1. 无条件极值（只给出了变量关系式，没有其他关系）
    2. 闭区域边界上的最值（变量关系式，和其他关系式）
    3. 闭区域上的最值（边界内为无条件极值，边界上为条件极值）

二重积分

概念

  1. 几何背景（曲顶柱体的体积）
  2. 性质
    1. 求区域面积
    2. 可积函数必有界
    3. 积分的线性性质（积分拆开）
    4. 积分的可加性
    5. 积分的保号性（积分的绝对值小于绝对值的积分）
    6. 二重积分的估值定理（积分的介值定理）
    7. 二重积分的中值定理
  3. 对称性
    1. 普通对称性（被积函数值在对称区间抵消）
    2. 轮换对称性（积分值和用什么字母表示无关，时常互换后前后相加）

计算

  1. 直角坐标系下的计算法
    1. 上下型（从上往下画直线，先交写上限，后交写下限）
    2. 左右型（从左往右画直线，先交写上限，后交写下限）
  2. 极坐标下的计算（在划定范围的时候可以直接将xy转换为rcos，rsin）
    1. 极点在区域之外
    2. 极点在区域边界上
    3. 极点在区域内部
  3. 极坐标系和直角坐标系的选择（满足以下使用极坐标）
    1. 是否为f(x^2+y^2)f(y/x)等形式
    2. 看积分区域是否为圆或者圆的一部分
  4. 极直互化
  5. 积分次序（当先求积分中存在不可积分时候，画图交换积分次序）
  6. 用二重积分处理一元积分的问题（累次积分和乘积：利用积分符号不变性）

常微分方程

微分方程的概念

  1. 微分方程（导数和自变量关系方程）
  2. 常微分方程（位置函数是一元函数）
  3. 微分方程的阶（方程中所含导数阶数）
  4. 微分方程的解（带入方程是方程成立的函数）
  5. 微分方程的通解（独立常数的个数等于微分方程的阶数）
  6. 初始条件和特解（确定常数的通解）

一阶微分方程的求解

  1. 可分离变量型（xy的微分可以直接移动到等式两边）
  2. 可化为变量可分离型
    1. dy/dx=f(ax+by+c)
    2. 齐次微分方程（可化为y/x）
  3. 一阶线性微分方程（y`+py=q）有通解公式，记住推导过程
  4. 伯努利方程

二阶可降阶微分方程的求解（y的二阶导，一阶导，y，x）

  1. 缺少y（替换一阶导，二阶导）
  2. 缺少x
    1. 一阶导替换，二阶导数替换为含p导数（变形为一阶线性微分方程）
    2. 通过通解公式求解得到一阶导
    3. 再对一阶导积分

高阶线性微分方程的求解

  1. 概念
    1. 二阶变系数线性微分方程，非齐次方程
    2. 二阶常系数微分方程
  2. 解的结构（二阶）
    1. 齐次方程的两个解线性组合为齐次方程的通解
    2. 非齐次方程的特解和齐次方程的通解组合为非齐次方程的通解
  3. 二阶常系数齐次线性微分方程的通解（记住公式）
  4. 二阶常系数非齐次线性微分方程的特解
  5. n阶常系数齐次线性微分方程的解

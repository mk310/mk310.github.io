一元函数积分学的概念和计算

不定积分

  1. 原函数和不定积分（不定积分没有上下限）
  2. 原函数（不定积分存在定理）
    1. 连续函数必有原函数
    2. 含有第一类间断点和无穷间断点的函数在包含该间断点的区间内没有原函数（含有震荡间断点的函数也不一定有原函数）

定积分

  1. 定积分的概念（将不规则图形划分为规则的梯形）

    • 积分区间有限
    • 被积函数有界

  2. 定积分存在定理

    1. 定积分存在的充分条件
      1. f在闭区间a,b上连续，原函数存在
      2. f在闭区间上单调，原函数存在
      3. f在闭区间上有界，且有有限个间断点（可以从间断点处拆开），原函数存在
    2. 定积分存在的必要条件
      1. 若定积分存在，其在上下界上有界

  3. 定积分的性质

    1. 求区间长度
    2. 积分中的常数可以提到积分符号之前
    3. 积分可加/拆分性
    4. 积分的保号性（积分的绝对值小于绝对值的积分）
    5. 估值定理（积分的介值定理）
    6. 积分中值定理

变限积分

  1. 变限积分的定义
  2. 变限积分的性质
    1. f在a,b上可积，则f的积分在a,b上连续（积分区域的增加是连续的）
    2. f在a,b上连续，则f的积分在a,b上可导（积分区域的增加是平滑的）
  3. 变限积分的求导公式

反常积分

  1. 反常积分的概念（1 积分区间无线，2 被积函数无界）

    • 被积函数越小，反常函数通常越容易收敛
    • 被积区间无线的反常函数收敛，其被积函数不一定收敛于0，可能是被积函数的无穷小程度超过了高的无穷大程度

  2. 无穷区间上反常函数的概念和敛散性

    1. 上积分区间无限
    2. 下积分区间无限
    3. 上下都无限

  3. 无界函数的反常积分概念和敛散性

    1. 积分后从被积函数的瑕点趋近

  4. 题型：

    1. 反常积分判敛公式

不定积分的积分法

  3. 换元法（当被积函数不容易积分的时候（反对幂函指三））

    1. 三角函数代换（三种平方形式）
    2. 恒等变形后做三角变换（先化成三角函数代换方式在代换）
    3. 根式代换（根式内无法凑平方的，直接将整个根式代换）
    4. 倒代换（被积函数分母的幂次比分子高两次或者以上的时候）

常微分方程

微分方程的概念

  1. 微分方程（导数和自变量关系方程）
  2. 常微分方程（位置函数是一元函数）
  3. 微分方程的阶（方程中所含导数阶数）
  4. 微分方程的解（带入方程是方程成立的函数）
  5. 微分方程的通解（独立常数的个数等于微分方程的阶数）
  6. 初始条件和特解（确定常数的通解）

一阶微分方程的求解

  1. 可分离变量型（xy的微分可以直接移动到等式两边）
  2. 可化为变量可分离型
    1. dy/dx=f(ax+by+c)
    2. 齐次微分方程（可化为y/x）
  3. 一阶线性微分方程（y`+py=q）有通解公式，记住推导过程
  4. 伯努利方程

二阶可降阶微分方程的求解（y的二阶导，一阶导，y，x）

  1. 缺少y（替换一阶导，二阶导）
  2. 缺少x
    1. 一阶导替换，二阶导数替换为含p导数（变形为一阶线性微分方程）
    2. 通过通解公式求解得到一阶导
    3. 再对一阶导积分

高阶线性微分方程的求解

  1. 概念
    1. 二阶变系数线性微分方程，非齐次方程
    2. 二阶常系数微分方程
  2. 解的结构（二阶）
    1. 齐次方程的两个解线性组合为齐次方程的通解
    2. 非齐次方程的特解和齐次方程的通解组合为非齐次方程的通解
  3. 二阶常系数齐次线性微分方程的通解（记住公式）
  4. 二阶常系数非齐次线性微分方程的特解
  5. n阶常系数齐次线性微分方程的解

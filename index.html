TypeScript

TypeScript（简称 TS）是由微软开发的开源编程语言，是 JavaScript 的静态类型超集。它在 JS 的基础上增加了类型系统，支持类型注解、接口、泛型和枚举等特性，能在代码编译阶段发现类型错误，提升代码的可读性与可维护性。TS 最终会被编译为纯 JavaScript，可在任何运行 JS 的环境（浏览器、Node.js）中执行。凭借强大的 IDE 智能提示和重构能力，TypeScript 已成为中大型前端项目的首选语言。

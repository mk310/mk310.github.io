表单域

常用属性

属性 属性值 作用
action url地址 指定接受处理表单数据的服务器程序的地址
method get/post 设置表单提交方式
name 名称 表单名称。区分一个页面中的多个表单域

表单元素

属性

属性值 描述
text 单行输入字段，默认宽度为20
submit 定义提交按钮，提交按钮会把表单中数据发送到服务端
password 密码字段，输入的字符被隐藏
radio 定义单选按钮
checkbox 定义复选框
Button 定义可点击按钮（用于通过javaScript启动脚本）
file 供上传文件
image 上传图像
reset 重置按钮
hidden 隐藏输入字段

标签属性

属性 属性值 描述
name 自定义 input元素的名称
value 自定义 input的值
checked checke 第一次夹在中间时被选中
maxlength 正整数 输入字符的最大长度
